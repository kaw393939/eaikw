# Design Guidelines

Purpose: Capture the core visual system and component rules for this portfolio.

## Tokens

- Colors: `--swiss-black`, `--swiss-white`, `--swiss-red`, `--swiss-gray-*`
- Spacing scale: `--space-sm` → `--space-5xl`
- Fluid typography: `--fluid-display`, `--fluid-h1`, `--fluid-h2`, etc.

## Components

- `.swiss-grid`, `.swiss-col-*` — layout grid
- `.btn-swiss`, `.btn-swiss-primary`, `.btn-swiss-accent` — button variants
- `.prose-swiss` — content typography container

## Usage notes

- Keep the palette minimal and use the red accent sparingly.
- Avoid decorative shadows and rounded corners unless justified for
  accessibility.
